LunarOS 2.0 is a comprehensive Linux distribution engineered exclusively for virtual 
machine environments. Built on the stable foundation of Ubuntu 22.04 LTS and featuring 
the lightweight, responsive Xfce desktop environment, this OS combines professional-grade 
stability with elegant moon-themed aesthetics.

DUAL VERSION ARCHITECTURE:
The single ISO includes two distinct versions, selectable during installation:

Lite Version (for older/minimal hardware):
  • Optimized for entry-level virtual machines
  • Requirements: NVIDIA GTX 950, 1-2GB RAM, 50GB storage
  • Includes: Essential utilities, Firefox, Xfce desktop, terminal utilities
  • Lightweight footprint with minimal resource consumption

Basic Version (for modern/powerful hardware):
  • Optimized for advanced users and developers
  • Requirements: NVIDIA RTX 3060+, 4GB+ RAM, 100GB+ storage
  • Includes: Full development stack, GIMP, Blender, LibreOffice, VS Code, VLC
  • Complete productivity and creative suite

KEY FEATURES:
✓ Ubuntu 22.04 LTS base (5 years of security updates)
✓ Xfce desktop environment (fast, responsive, customizable)
✓ Pre-configured Firefox browser
✓ Dual-version single ISO (space-efficient distribution)
✓ Moon-themed aesthetic (custom wallpapers, icons, color scheme)
✓ 5-Layer Safety Warning System:
  1. GRUB boot menu warning (cannot bypass)
  2. Login screen warning (/etc/issue.net)
  3. Terminal welcome message (/etc/motd)
  4. Installer confirmation (5 stages requiring explicit "I UNDERSTAND" confirmations)
  5. File deletion protection (safe-rm utility for critical system files)
✓ Fast boot times (15-30 seconds)
✓ Low idle memory usage (~300MB)
✓ Comprehensive documentation

CRITICAL DESIGN CONSTRAINT:
LunarOS 2.0 is designed exclusively for virtual machines (VirtualBox, KVM, VMware, 
Hyper-V, Proxmox). The five-layer warning system ensures users understand this VM-only 
requirement. Installing on real hardware is NOT reccomended and will result in system 
instability.
